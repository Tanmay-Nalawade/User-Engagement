# Frontend

This UI is meant to run **in Docker**, not with local `npm run dev`.

From the **project root** (`User-Engagement/`):

```bash
docker compose up --build
```

Open **http://localhost:3000**

---

Local dev (optional — requires install first):

```bash
npm install
npm run dev
```

Open http://localhost:5173 (API must be on :8080).
